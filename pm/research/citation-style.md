# Citation Style

## Internal Architecture Citations

Use the local architecture file and line ranges when citing plan constraints or sequencing. Example:

- `Architecture_Plan.md lines 97-109`
- `Architecture_Plan.md lines 413-426`

## External Citations

Use the official URL and the access date if the source was materially used in planning or implementation. Example:

- `OpenSearch document-level security docs, accessed 2026-03-06: https://docs.opensearch.org/latest/security/access-control/document-level-security/`

## Planning Rule

- Do not cite secondary summaries when a primary source exists.
- If a task depends on external research, add the source to [`../08_research_register.md`](../08_research_register.md).
