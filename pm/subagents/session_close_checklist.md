# Session Close Checklist

- Update task status in the relevant backlog file
- Update [`../11_status_dashboard.md`](../11_status_dashboard.md) if active queue or blocker state changed
- Update [`../06_risk_register.md`](../06_risk_register.md) if a new risk appeared or a risk changed
- Update [`../07_decision_log.md`](../07_decision_log.md) if a new decision was made
- Update [`../08_research_register.md`](../08_research_register.md) if external research was used
- Update [`../14_artifact_inventory.md`](../14_artifact_inventory.md) if artifact definitions, systems of record, or promotion rules changed
- Update [`../15_non_functional_requirements.md`](../15_non_functional_requirements.md) if cross-cutting quality requirements changed
- Append or update the session log under [`../logs/`](../logs)
- Leave a complete handoff using [`handoff_template.md`](./handoff_template.md)
